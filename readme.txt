3.頑張った機能やポイント
私なりにではありますが・・・
殆どがコピペではありますが、完成まで30時間ほどかかりました。
アニメーションやレスポンシブル等のサイズ調整は行っておりませんが、ホバーやページ内リンクなど、仕様書をどこまで再現できるか気にして作成いたしました。


4.課題の感想
現状はGoogle検索やコピペに頼っていることが殆どです。
CSSをググってコピペして使用しても、数日後（なんなら数時間後）にそのCSSを忘れてしまい、何度も同じワードで検索を繰り返している状態。
理解できているかどうかも自分ではわかりませんが、全体的な構造や大枠を決め、表装を行う段取りが良いと考え、最初はHTMLを組み、その後にCSSの作業を行いました。
CSSが反映されない場合の対処がわからず、何度かセクション全体を削除して作り直したこともありますので、エラー時の最善の解決方法が知りたいです。
自分の中では全く理解出来ていないのかなと思っており、先行き不安ではあります。
が、泣き言やマイナスなことばかりいっても何も解決しないのは自分でわかってますので、復習と次回の予習に勤しみます。
また、私１人では間違えなく挫折していたかと思います。
